# C-Proxy-Server
proxy server implemented using c language

###프록시서버 실행
1. proxy_server.c 내에 SET SERVER ADDRESS!! 과 SET PORT NUMBER 주석처리된 부분을 server의 ip 주소와 포트번호로 바꿔준다.
2. $ make --> compile
3. $ ./proxy_server

###웹 브라우저로 테스트
1. 웹 브라우저 실행. 프록시 서버를 설정해준다. ip 주소, 포트 번호
